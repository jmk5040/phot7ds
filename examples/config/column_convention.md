# Catalog column & FITS header conventions

This file documents the names you'll see in the catalogs produced by
`phot7ds.run_photometry`. Use it as a quick reference when wiring
downstream consumers.

## Aperture labels

Aperture column names are always **zero-padded** to two digits:

| Aperture diameter | Column prefix |
|-------------------|---------------|
| 5"                | `aper05_`     |
| 10"               | `aper10_`     |
| Kron auto         | `auto_`       |
| Kron auto-corr.   | `autoc_`      |

So flux/mag columns look like `aper05_mag_m475`, `aper10_mag_err_g`,
`auto_flags_z`.

## Magnitude / flux columns

```
{aperture}_{quantity}_{band}
```

Where:

* `aperture` ∈ `aper05`, `aper10`, `auto`, `autoc`
* `quantity` ∈ `flux`, `flux_err`, `mag`, `mag_err`, `flags`
* `band` is the lowercase filter name (`g`, `r`, `i`, `m400`, …)

The band is always the **last** token in the column name.

## Calibrated columns

After zero-point calibration, each measurement column carries a
companion ZP-corrected column generated by
`phot7ds.calibration.calibrate_zeropoints`. The constant per-(aperture,
band) ZP and its scatter also land in the FITS header (see below).

## Catalog file naming

`run_photometry` writes two FITS files per run:

| File                       | Contents                              |
|----------------------------|---------------------------------------|
| `{run_name}_raw.fits`      | Raw SE++ output (pre-calibration).    |
| `{catalog_name}` / `_path` | Final calibrated catalog.             |

The final catalog basename is whatever you pass to `catalog_name` /
`catalog_path` — no `_phot.zp.fits` suffix is forced. If you supply a
stem without `.fits`, the extension is appended.

## FITS header keys

### Calibration (per aperture / band)

| Key            | Meaning                              | Unit |
|----------------|--------------------------------------|------|
| `ZP{AP}M{BND}` | Constant zero-point                  | mag  |
| `ZE{AP}M{BND}` | 1-σ scatter of the ZP fit            | mag  |

`{AP}` is two chars (`05`, `10`, `AU`, `AC`); `{BND}` is one char for
broadbands (`G`/`R`/`I`/`Z`) and three digits for medium-bands
(`400`…`875`).

### Depth (5" aperture only)

| Key             | Meaning                                       | Unit |
|-----------------|-----------------------------------------------|------|
| `UL{N}EM{BND}`  | N-σ depth from the magnitude-error curve fit  | mag  |
| `UL{N}RM{BND}`  | N-σ depth from empty apertures / bkg RMS      | mag  |
| `BRMSM{BND}`    | Empty-aperture sigma (`σ_ADU`)                | ADU  |

`{N}` is the SNR threshold (typically `5`).

### Provenance / coverage

| Key        | Meaning                                          |
|------------|--------------------------------------------------|
| `PHOTVER`  | phot7ds package version                          |
| `PHOTRUN`  | Run name (stem of this catalog)                  |
| `PHOTDATE` | Catalog write timestamp (UTC, ISO-8601)          |
| `PHOTUSR`  | Username that produced the catalog               |
| `PHOTHOST` | Hostname that produced the catalog               |
| `PHOTPY`   | Python interpreter version                       |
| `DETLABEL` | `'DELVE'` or `'7DT'`                             |
| `DETIMG`   | Detection image basename                         |
| `COVMASK`  | Coverage mask basename                           |
| `BADPMASK` | Bad-pixel mask basename (if any)                 |
| `REFCAT`   | Reference catalog basename                       |
| `NSCIIMG`  | Number of measurement images                     |
| `SCIMGNNN` | Per-image basename (`NNN` = zero-padded index)   |
| `MSKRATIO` | Ratio of pixels masked in the coverage mask     |
| `DETTHR`   | SE++ detection threshold (σ)                     |
| `DETMINAR` | SE++ detection minimum area (pix)                |
| `KRNMINR`  | SE++ auto-kron minimum radius (pix)              |
| `PARTMINC` | SE++ partition minimum contrast                  |
| `FIXAPER`  | Fixed apertures (arcsec, comma-separated)        |
| `PIXSCALE` | Pixel scale (arcsec/pix)                         |
